# 🎬 Audition Helper AI

An AI-powered audition analysis tool for actors.

Audition Helper AI analyzes scripts, casting briefs, character descriptions and audition sides to help actors prepare better self-tapes and performances.

## Features

* Character Breakdown
* Scene Analysis
* Objective & Obstacle Detection
* Subtext Analysis
* Emotional Arc Analysis
* Character Animal Suggestion
* Casting Director Notes
* Quick Preparation Tips
* Multiple Analysis Styles
* Download Analysis as TXT

## Built With

* Python
* Streamlit
* Google Gemini API
